# 退货退款 Skill

## 触发条件
当用户明确表达"我要退货""申请退款""帮我退"等正式申请意图时激活。

## 参数说明
| 参数 | 必填 | 说明 |
|------|------|------|
| order_id | 是 | 订单号 |
| reason | 否 | 退货原因 |
| item | 否 | 商品名称 |

## 执行流程
1. 确认用户确实要申请退货（而非咨询规则）
2. 调用 `request-return` 提交退货申请
3. 告知用户退货单号和预计退款时间（1-3 个工作日）

## 注意事项
- **严格区分**：用户咨询退货规则/政策 → 用 `knowledge_search`，不要调用本工具
- 必须确认 order_id 存在，否则先让用户提供
- 退款 1-3 个工作日原路返回
